Shopping Cart (JavaScript)

📌 Project Overview

This is a Shopping Cart web application developed as part of the Udacity Scholarship Program.

The project focuses on implementing core e-commerce cart functionality using JavaScript. The HTML and CSS structure were provided by Udacity, while all interactive behavior and logic were implemented in JavaScript.

The goal of this project is to demonstrate understanding of DOM manipulation, event handling, and dynamic state updates in a real-world-like shopping cart system.

⸻

✨ Features

* Add products to the shopping cart
* Remove products from the cart
* Increase or decrease item quantities
* Automatically calculate total price
* Real-time updates to the UI using DOM manipulation

⸻

🛠️ Technologies Used

* HTML (provided by Udacity)
* CSS (provided by Udacity)
* JavaScript (custom implementation)

⸻

🧠 What I Implemented

* Built all shopping cart functionality using vanilla JavaScript
* Managed cart state (items, quantities, totals)
* Implemented dynamic updates to the DOM without page reloads
* Handled user interactions such as add/remove/update quantity
* Ensured correct calculation of totals based on cart changes

⸻

🚀 How to Run the Project

1. Clone or download the repository
2. Open the project folder
3. Open index.html in any modern web browser

No additional dependencies or setup required.
